## 🚀 Embark on Your Techbook Journey

1. **Clone the Innovation**

```shell
git clone https://github.com/yourusername/techbook.git
cd techbook
```

2. **Fuel the Frontend**

```shell
cd client
npm install
```

3. **Empower the Backend**

```shell
cd ../server
npm install
```

4. **Set the Stage**  
   Create a `.env` file in the `server` directory:

```plaintext
MONGO_URI=your_mongodb_connection_string
FIREBASE_API_KEY=your_firebase_api_key
STRIPE_SECRET_KEY=your_stripe_secret_key
```

5. **Ignite the Engines**

```shell
# Launch the backend
cd server
npm start

# Activate the frontend
cd ../client
npm start
```

🎉 Techbook awaits at [http://localhost:3000/](http://localhost:3000/)
